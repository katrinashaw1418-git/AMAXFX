## 🔧 2. ADD ONE CRITICAL TABLE (BIGGEST WIN)

### ✅ Add: `transactions` (unified audit trail)

```ts
export const transactions = pgTable("transactions", {
  id: serial("id").primaryKey(),

  userId: integer("user_id"),

  type: text("type"), 
  // deposit, withdrawal, fx, crypto_buy, crypto_sell, transfer

  assetType: text("asset_type"), 
  // fiat | crypto

  asset: text("asset"), 
  // AUD, USD, BTC

  amount: decimal("amount"),

  direction: text("direction"), 
  // in / out

  status: text("status"),

  referenceId: text("reference_id"),

  createdAt: timestamp("created_at").defaultNow(),
});
```

---

### 🧠 Why this alone is powerful

This gives you:

* Full audit trail ✅
* Transaction monitoring base ✅
* AUSTRAC traceability ✅

👉 Without rebuilding your system

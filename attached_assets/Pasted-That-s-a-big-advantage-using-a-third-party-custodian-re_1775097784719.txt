# 🧩 Required code-level patches

## 1. 🔴 Idempotency (CRITICAL)

Custodian APIs + retries = duplicate risk

```ts
const idemKey = req.header("Idempotency-Key");
if (!idemKey) throw new Error("Missing Idempotency-Key");

const existing = await storage.getIdempotentResult(userId, idemKey);

if (existing) return res.json(existing);

// process transaction once
const result = await processTransaction();

await storage.saveIdempotentResult(userId, idemKey, result);

return res.json(result);
```

## 3. 🔴 Custodian reconciliation (MOST IMPORTANT)

```ts
async function reconcileWithCustodian(userId: number) {
  const internal = await storage.getWallets(userId);
  const external = await custodianApi.getBalances(userId);

  const mismatches = internal.filter((w) => {
    const ext = external.find(e => e.currency === w.currency);
    return Math.abs(Number(w.balance) - Number(ext.balance)) > 0.01;
  });

  if (mismatches.length > 0) {
    console.error("custodian_mismatch", mismatches);
  }

  return mismatches;
}
```

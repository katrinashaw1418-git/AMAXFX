 It sounds like the issue is that when the user inputs an investment amount and clicks the "Invest Now" button, the balance of the capital invested isn’t reflecting that change. This could be a problem in how the event is handled or how the state of the capital invested is updated.

I’ll outline a general way to fix this, assuming you're using JavaScript (for a front-end solution). If you're working with another language or framework, just let me know!

Here's a simple approach in plain JavaScript:
HTML:

Have an input field where the user can enter the investment amount.

Display the balance of capital invested somewhere in the page.

A button to trigger the "Invest Now" action.

JavaScript:

Use a function to handle the investment logic when the button is clicked.

Update the displayed capital invested based on the user's input.

// Initial capital invested (starting from zero)
let capitalInvested = 0;

// Reference to DOM elements
const investmentAmountInput = document.getElementById('investmentAmount');
const capitalInvestedElement = document.getElementById('capitalInvested');
const investNowButton = document.getElementById('investNowBtn');

// Function to update the capital invested
function updateCapitalInvested() {
    // Get the value from the input field
    const investmentAmount = parseFloat(investmentAmountInput.value);

    // If a valid number is entered, add it to the capital invested
    if (!isNaN(investmentAmount) && investmentAmount > 0) {
        capitalInvested += investmentAmount;
        capitalInvestedElement.textContent = capitalInvested.toFixed(2);  // Display updated amount
        investmentAmountInput.value = ''; // Clear the input field after investment
    } else {
        alert('Please enter a valid amount');
    }
}

// Add event listener to the "Invest Now" button
investNowButton.addEventListener('click', updateCapitalInvested);
How it works:
HTML Structure:

You have an input field (investmentAmount) for the user to enter their investment amount.

There’s a button (investNowBtn) that the user clicks to invest.

The current capital invested is displayed inside the <span id="capitalInvested">.

JavaScript Logic:

capitalInvested keeps track of the total capital invested. It starts at 0.

When the user clicks the "Invest Now" button, the updateCapitalInvested() function is called.

It retrieves the value from the input field and checks if it's a valid number.

If valid, it adds that amount to capitalInvested and updates the display (capitalInvestedElement).

If the input is invalid (empty or non-numeric), an alert is shown.

What to Check:
Make sure that the button's click event is properly wired up to the function (updateCapitalInvested).

Ensure the investment amount is being correctly added to the capitalInvested variable and that it's being displayed in the DOM (capitalInvestedElement).

Clear any potential errors in the console (like incorrect variable names or event binding).

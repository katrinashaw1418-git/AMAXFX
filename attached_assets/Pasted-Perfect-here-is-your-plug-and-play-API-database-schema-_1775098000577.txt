# 🧩 🗄️ 1. CORE DATABASE SCHEMA (PRODUCTION-READY)

## 🔴 USERS

```sql
CREATE TABLE users (
  id SERIAL PRIMARY KEY,
  email TEXT UNIQUE NOT NULL,
  password_hash TEXT NOT NULL,
  mfa_enabled BOOLEAN DEFAULT FALSE,
  kyc_status TEXT DEFAULT 'pending', -- pending | approved | rejected
  created_at TIMESTAMP DEFAULT NOW()
);
```

---

## 🔴 WALLETS

```sql
CREATE TABLE wallets (
  id SERIAL PRIMARY KEY,
  user_id INT REFERENCES users(id),
  currency TEXT NOT NULL,
  balance NUMERIC(18,6) DEFAULT 0,
  UNIQUE(user_id, currency)
);
```

---

## 🔴 LEDGER (SOURCE OF TRUTH)

```sql
CREATE TABLE ledger_entries (
  id SERIAL PRIMARY KEY,
  user_id INT,
  type TEXT, -- deposit, withdrawal, exchange
  currency TEXT,
  amount NUMERIC(18,6),
  reference TEXT,
  metadata JSONB,
  created_at TIMESTAMP DEFAULT NOW()
);
```

---

## 🔴 TRANSACTIONS (USER VIEW)

```sql
CREATE TABLE transactions (
  id SERIAL PRIMARY KEY,
  user_id INT,
  type TEXT,
  status TEXT,
  amount NUMERIC(18,6),
  currency TEXT,
  fx_rate NUMERIC(18,8),
  fee NUMERIC(18,6),
  reference TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

---

## 🔴 SYSTEM EVENTS (AUDIT)

```sql
CREATE TABLE system_events (
  id SERIAL PRIMARY KEY,
  type TEXT,
  user_id INT,
  payload JSONB,
  created_at TIMESTAMP DEFAULT NOW()
);
```

---

## 🔴 PRICE SNAPSHOTS

```sql
CREATE TABLE price_snapshots (
  id SERIAL PRIMARY KEY,
  asset TEXT,
  price NUMERIC(18,8),
  source TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);
```

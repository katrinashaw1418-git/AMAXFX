Those fixes are aligned with the earlier issues you were addressing around synthetic history, source-aware labels, and shared valuation logic.

## Biggest remaining issue

The biggest remaining model problem is:

**investment history is date-aware, but wallet history is not.**

So if the portfolio graph still looks wrong, that is probably why.

Right now:

* investments are valued as of the requested date
* wallets are mostly valued using today’s balances

That means the total portfolio history is still only partly historical.

## Concrete example of the issue

This function is the problem area:

```ts
async function calculatePortfolioTotalsAtDate(userId: number, asOfDate: Date = new Date()) {
  const wallets = await storage.getWallets(userId);
  ...
}
```

It fetches current wallets, not wallet balances as of `asOfDate`.

So even if `asOfDate` is one year ago, it still uses today’s wallet balances.

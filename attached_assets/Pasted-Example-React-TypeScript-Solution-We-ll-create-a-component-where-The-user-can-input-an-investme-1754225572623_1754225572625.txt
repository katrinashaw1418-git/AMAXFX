Example: React + TypeScript Solution
We’ll create a component where:

The user can input an investment amount.

When they click "Invest Now," the capital invested will update.

The updated capital will be displayed in real time.

Steps:
State Management: Use React's useState hook to track the capital invested.

Input Handling: Capture the input value when the user types in the investment field.

Button Click: When the "Invest Now" button is clicked, update the capital invested based on the input.

Code Example:
InvestmentComponent.tsx
tsx

Edit
import React, { useState } from 'react';

const InvestmentComponent: React.FC = () => {
    // State for the investment amount input and capital invested
    const [investmentAmount, setInvestmentAmount] = useState<number>(0);
    const [capitalInvested, setCapitalInvested] = useState<number>(0);

    // Handle input change
    const handleInputChange = (event: React.ChangeEvent<HTMLInputElement>) => {
        const value = parseFloat(event.target.value);
        if (!isNaN(value)) {
            setInvestmentAmount(value);
        }
    };

    // Handle Invest Now button click
    const handleInvestClick = () => {
        if (investmentAmount > 0) {
            setCapitalInvested(prevCapital => prevCapital + investmentAmount);
            setInvestmentAmount(0); // Reset input field after investment
        } else {
            alert('Please enter a valid investment amount');
        }
    };

    return (
        <div>
            <h1>Investment Dashboard</h1>
            
            <div>
                <label htmlFor="investmentAmount">Enter Investment Amount: </label>
                <input
                    type="number"
                    id="investmentAmount"
                    value={investmentAmount}
                    onChange={handleInputChange}
                    placeholder="Enter amount"
                />
            </div>

            <button onClick={handleInvestClick}>Invest Now</button>

            <h2>Capital Invested: ${capitalInvested.toFixed(2)}</h2>
        </div>
    );
};

export default InvestmentComponent;
Explanation:
State Management:

We use useState to manage two pieces of state:

investmentAmount: Tracks the input value for the investment.

capitalInvested: Tracks the total capital invested.

Input Handling:

handleInputChange is called every time the user types in the input field. It updates the investmentAmount state with the new value (only if it's a valid number).

Invest Now Button:

handleInvestClick is called when the user clicks the "Invest Now" button.

It adds the investmentAmount to the capitalInvested state and resets the investmentAmount input to 0.

Display:

The capital invested is displayed using capitalInvested.toFixed(2) to ensure it's shown as a decimal with two places.

TypeScript Notes:
Type Annotations: We use TypeScript's type annotations for the useState hooks (number) to ensure type safety.

Event Types: We also specify the correct event type (React.ChangeEvent<HTMLInputElement>) for the input change event.

Final Notes:
Edge Case Handling: The input and click handler checks that the user entered a valid number and prevents invalid or negative investments.

Reactivity: Because React’s state updates trigger re-renders, once you update capitalInvested or investmentAmount, the component will automatically reflect the new values.

Debugging Tips:
Check Console for Errors: If the capital isn't updating, check the browser console for any JavaScript/TypeScript errors.
